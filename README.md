
# Digital Financial Innovation and Monetary Policy Transmission  in West Africa: Panel Evidence from ECOWAS Economies

## Overview
This repository contains the code and data used in the paper:
**"Digital Financial Innovation and Monetary Policy Transmission 
in West Africa: Panel Evidence from ECOWAS Economies"**

## Repository Structure
panel_clean.csv - Dataset
panel_final.py - Analysis script

## Requirements
- Software: Python 
- Key packages: numpy, pandas, matplotlib, seaborn, requests

## How to Run
1. Clone the repository
2. Install dependencies
3. Run scripts in `code/` folder
